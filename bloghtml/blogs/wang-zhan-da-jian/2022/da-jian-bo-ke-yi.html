    VuePress 搭建博客（一）vuepress-theme-reco 主题博客初始化

    避海拦天 2022/3/19 VuePressvuepress-theme-reco博客搭建

    VuePress 是一款 Vue 驱动的静态网站生成器，以 Markdown 为中心的项目结构，以最少的配置帮助你专注于写作。

    vuepress-theme-reco 是一款简洁而优雅的 vuepress 博客&文档主题。

    效果：

    # 零、准备工作

    1. 安装 Yarn

      npm install --global yarn
      1

      如果报错，需要用管理员权限

      sudo npm install --global yarn
      1

      安装后检查是否安装成功

      yarn --version
1.22.18
      1
      2

      如果没有Yarn，也可以用npm或者npx等其他包管理器，但是VuePress官网推荐使用Yarn来安装，因为在一些情况下，npm会生成错误的依赖树，在后期想要给网站添加一些插件的时候会出现错误。

    2. 下载 vuepress-reco/theme-cli

      新建一个空项目目录Blog(起什么名字随意)

      在空目录下运行命令，初始化包管理器。

      yarn init #或 yarn init -y 默认所有选项为yes
      1

      在目录下运行命令，将VuePress、vuepress-reco/theme-cli 安装为本地依赖。很多教程使用的命令是带有--global的全局安装，但是VuePress官网已经不再推荐使用全局安装了，所以使用-D在目录本地安装。

      yarn add -D @vuepress-reco/theme-cli
      1

      安装后目录下会自动生成一个node_modules文件夹和两个文件

      node_modules  package.json  yarn.lock
      1

    # 一、初始化博客

    1. 使用 theme-cli初始化博客文件

      在Blog目录下运行初始化命令yarn theme-cli init

      会出现很多初始化的问题，可以全回车跳过，之后再配置，也可以根据问题输入想要的配置。建议大家直接跳过，先把项目跑起来之后，可以慢慢修改。

      问题1 是否新建一个文件夹，我选择的是No。如果选择Yes 下个问题会问你新文件夹的名称。

      问题2 项目的标题（可回车直接跳过）

      问题3 项目的描述 （可回车直接跳过）

      问题4 作者的名字 （可回车直接跳过）

      问题5 你想要主页是什么风格的，有blog/doc/2.x三个选项，选择blog博客风格，如果做的是文档可以选择doc风格，2.x是个alpha的全新版本，也可以尝鲜。这里我选择的是blog。选择之后会开始下载模板文件，稍等一会儿。

      初始化结束后，目录下会多出一个blogs文件夹，用来存放你的博客文章，docs文件夹存放文档类文章，.vuepress文件夹，存放博客的配置文件，图标背景图片等资源。README.md文件，是你的博客主页。如果看不到.vuepress文件夹，在文件夹设置中打开“显示隐藏文件”。

      blogs  .gitignore    package.json  .vuepress
docs   node_modules  README.md     yarn.lock
      1

    2. 安装需要的Package

      在Blog目录下直接运行命令

      yarn install
      1

      或

      npm install
      1

    # 二、 看看效果

    1. 在Blog目录下运行命令

      yarn dev
      1

      等待一会后会出现进度条，结束后会出现成功的提示

      在浏览器中打开 http://localhost:8080/ (如果端口被占用会变成8081或8082依次往上)

    # 四、自定义首页配置

    1. 打开Blog目录下的README.md对首页的配置进行更改。

    2. ---
home: true
heroText: 避海拦天的学习笔记 # 展示在主页正中的标题
tagline: Things change, roll with it. # 展示在标题下方的文字
# heroImage: /hero.png
# heroImageStyle: {
#   maxWidth: '600px',
#   width: '100%',
#   display: block,
#   margin: '9rem auto 2rem',
#   background: '#fff',
#   borderRadius: '1rem',
